## 🔗 Cara Cari GIF Transparan di Giphy:

### 1. Search Keywords:
- `"transparent sticker"`
- `"cute sticker"`
- `"transparent gif"`
- `nama karakter + "sticker"`

### 2. Filter di Giphy:
- Setelah search → Klik "Filters"
- Pilih type: **Stickers** (ini biasanya transparan)

### 3. Cek Manual:
- Klik GIF
- Lihat background - kalau ada pattern kotak-kotak (checkerboard) = transparan
- Klik **Share** → **Copy GIF Link**

## ⚠️ Hindari:
- Tenor direct URLs (sering punya background)
- GIF dengan background solid/putih
- Large GIFs (>500KB)

## 💡 Tips:
- Pilih GIF dengan ukuran 100-300px untuk main character
- Pilih 20-50px untuk decorations
- Test GIF di browser dulu sebelum pakai
